Introduction to CSE211 Web Programming

This course introduces students to the principles and practices of modern web application development. The curriculum covers the complete lifecycle of building web applications, from design and front-end implementation with HTML5 and CSS3, to adding interactivity with JavaScript, and developing dynamic front-end applications using React.

On the server side, students will use Node.js and the Express.js framework to design and implement RESTful APIs, integrate with relational databases (SQLite/Postgres), and manage data-driven workflows. Emphasis is placed on responsive design, accessibility, security, and industry-standard development tools and practices.

"By the end of the course, students will have built and deployed a full-stack web application to the cloud, gaining both practical skills and real-world project experience."

Course Objectives (Student Learning Outcomes)

  1. Design and develop responsive, and semantically structured websites using modern HTML5 and CSS3, including Flexbox and Grid layouts.
  2. Implement interactivity with modern JavaScript (ES6+), including DOM manipulation, events, form validation, and API integration.
  3. Build front-end applications using React with components, state management, and client-side routing.
  4. Develop server-side applications and RESTful APIs using Node.js and the Express framework.
  5. Integrate databases (SQLite/Postgres) with web applications to perform Create, Read, Update, and Delete (CRUD) operations.
  6. Apply web standards including HTTP fundamentals, WCAG 2.2 accessibility guidelines, and web security practices (HTTPS, CORS, JWT).
  7. Use modern development tools and workflows, including Git/GitHub, VS Code, and deployment pipelines.

Topics Covered

  • Core Tools: HTML5, CSS3, JavaScript, React, Express.js.
  • Backend & Database: Node.js, SQLite, PostgreSQL.
  • Version Control: Git & GitHub.
  • Performance: Core Web Vitals and accessibility audits.
